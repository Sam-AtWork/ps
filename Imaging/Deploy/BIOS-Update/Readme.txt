Folder Structure :
Manufacturer / Model (exact Model from WMI - 'wmic csproduct get name')

New Model Addition / Update BIOS :
Place BIOS executable in correct folder structure (single executable only)
Create a '.ver' file named after the added BIOS version (eg. A13.ver)

Note : Always read BIOS notes carefully
BIOS Updates may require incremental updates (not currently implemented)
